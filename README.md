## 📥 如何导入书源到厚墨？

1. 打开 **厚墨 App**
2. 进入「书源管理」→「导入」
3. 选择以下任一方式：
   - **复制 JS 代码** →「从剪贴板导入」
   - **输入 Raw 链接**（如 GitHub/Gitee 的 raw URL）→「从网络导入」

> 💡 推荐使用 **Raw 链接**，便于后续自动更新。
